## Why

The repository now ships example code intended for reuse by both proprietary and open-source consumers. That reuse story should be explicit rather than implied.

## What Changes

- Add an MIT `LICENSE` file.
- Declare MIT licensing in the README.
- Declare MIT licensing in package metadata.

## Impact

- Removes ambiguity for downstream users of the example code.
- Keeps reuse friction low for commercial and open-source adopters.
